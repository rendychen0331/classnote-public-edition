- 學期切換：deleteSemester(old) → insertCourse(new)（覆蓋語意）
- 臨時修改：`CourseOverrideEntity`，overrideType = "cancel" | "replace"
- 提醒多次通知：每筆 `ReminderNotificationEntity` 對應一個 AlarmManager alarm
- 完成消失：`markCompleted(id)`，list 只查 `isCompleted=0`
- Navigation Safe Args：courseId:Long(-1=新增), dayOfWeek:Int(1), period:Int(1); reminderId:Long(-1=新增)
- CourseEditFragment：AutoCompleteTextView + ExposedDropdownMenu（MD3 風格）
- WeekViewFragment：動態 TextView cells，空格點擊帶入 dayOfWeek/period 跳轉新增課程
- ViewModel 用 Factory pattern 注入 Repository
